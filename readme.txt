Collapse fields
-------------------------------------------------------------------------------

Version: 1.0.0
Author: Guillem Lorman <guillem@bajoelcocotero.com>
Build Date: 05 MArch 2010
Github Repository: 
Requirements: Symphony 2

Add a link to collapse the specified fields in the publish page, making possible to the user to show or hide a group of fields.

Installation
-------------------------------------------------------------------------------

1. Upload the 'collapse_field' folder in this archive to your Symphony
   'extensions' folder.

2. Enable it by selecting the "Field: Collapse Fields", choose Enable from the
   with-selected menu, then click Apply.

3. The field will be available in the list when creating a Section.

Usage
-------------------------------------------------------------------------------

When adding this field to a section, the following options are available to you:

* **Title** is the text showed as a collapse link to show or hide the field set
* **Number of fields** number of fields to be collapsed when clic the link. For example, set a value of 3 will collapse next 3 fields.
